]# To-Do App 📋

A simple and responsive To-Do app built with React and Vite. Users can add, edit, and delete tasks with local storage support.

## 🛠 Tech Stack
- React.js
- Vite
- CSS

## 🚀 Features
- Add, edit, and delete tasks
- Data persists using `localStorage`
- Responsive UI
